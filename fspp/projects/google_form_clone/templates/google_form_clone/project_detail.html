Project Overview

This project is a complete front-end clone of a Google Form, built from scratch to replicate its distinct visual style and user experience. The goal was to practice semantic HTML and advanced CSS, including the use of CSS variables for theming and a responsive layout that works across all devices.

Key Features

  • Authentic Styling: Meticulously crafted CSS to match Google's color palette, typography, and layout.

  • Interactive Elements: Includes focus states on input fields and hover effects on buttons, just like the original.

  • Component-Based Structure: The form is built with modular "cards" for each question, making it easy to add or remove sections.

  • Fully Responsive: The layout adapts seamlessly from mobile phones to desktop screens.

Technology Stack

The following technologies were used for the frontend:

  • HTML5
  • CSS3
  • Flask & Jinja2
