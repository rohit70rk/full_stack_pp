Project Overview

This project demonstrates how to Embed a live Google Form directly into a Flask web application. The goal was to integrate external services seamlessly while maintaining a consistent design and user experience within the portfolio site.

Key Features

  • Live Google Form Integration: An actual Google Form is embedded into the site using an iframe.

  • Responsive Layout: The iframe container is styled to adapt well across desktop and mobile devices.

  • Consistent UI: Custom CSS ensures the embedded form blends with the overall design of the portfolio.

  • Practical Use Case: A quick and effective way to collect data (e.g., surveys, feedback, student info) without building a backend form handler.

Technology Stack

The following technologies were used to build this project:

  • Python
  • Flask & Jinja2
  • HTML5
  • CSS3
  • Google Forms
