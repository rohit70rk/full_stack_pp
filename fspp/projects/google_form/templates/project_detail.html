Project Overview

The primary goal of this project was to understand the fundamentals of web form handling. It involves creating a user-friendly interface for students to submit their information, validating the data on the backend to ensure accuracy, and preparing it for storage in a database.

Key Features

  • Dynamic Form Generation: The form is rendered using Jinja2 templates, allowing for easy modifications.

  • Input Validation: Server-side checks are implemented to validate fields like email, name, and student ID before processing.

  • Data Submission Handling: A dedicated Flask route processes the submitted form data using `POST` requests.

  • Responsive Design: The form is designed to be fully functional on both desktop and mobile devices.

Technology Stack

The following technologies were used to build this project:

  • Python
  • Flask
  • HTML5
  • CSS3
  • Jinja2
